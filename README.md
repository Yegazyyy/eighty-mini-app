# Eighty v2.3.0

Eighty — современный Telegram Mini App для ведения дневника питания, подсчёта калорий и КБЖУ, контроля воды, веса и анализа прогресса.

## Возможности

### Дневник

- добавление продуктов в `Завтрак`, `Обед`, `Ужин` и `Перекус`;
- контроль калорий за день;
- контроль белков, жиров и углеводов;
- контроль воды с быстрым добавлением;
- быстрый поиск продуктов;
- компактные карточки продуктов внутри приёмов пищи.

### Добавление

- создание собственных продуктов;
- создание собственных блюд;
- создание шаблонов приёмов пищи из уже заполненного дневника;
- встроенная база продуктов Eighty;
- единый поиск по личным продуктам, блюдам и встроенной базе;
- поддержка трёх типов продуктов:
  - `На 100 грамм`;
  - `Поштучно`;
  - `После приготовления`.

### Аналитика

- статистика за день;
- просмотр рациона за выбранный день;
- история дней;
- средние значения КБЖУ;
- история воды.

### Продукты

- личная библиотека продуктов;
- личная библиотека блюд;
- поиск по пользовательским продуктам, блюдам и встроенной базе Eighty;
- редактирование продуктов и блюд;
- отображение продуктов встроенной базы Eighty.

### Профиль

- автоматический расчёт КБЖУ;
- ручной расчёт КБЖУ;
- расчёт BMR по формуле Миффлина — Сан Жеора;
- история веса;
- достижения;
- удаление аккаунта.

## Структура приложения

- `Дневник` — ежедневный рацион, калории, КБЖУ, вода и приёмы пищи.
- `Аналитика` — статистика, история дней, рацион за выбранную дату и динамика воды.
- `Добавить` — выбор продуктов, встроенная база Eighty, создание продуктов, блюд и шаблонов.
- `Продукты` — личная библиотека продуктов и блюд, поиск, редактирование и встроенная база.
- `Профиль` — параметры пользователя, расчёты, вес, достижения и управление аккаунтом.

## Запуск

```powershell
npm.cmd run dev
```

Откройте [http://localhost:3000](http://localhost:3000).

## Telegram

Для Telegram Mini App нужен публичный HTTPS URL, который проксирует этот сервер. Если задан `TELEGRAM_BOT_TOKEN`, сервер проверяет подпись `initData`. Серверные уведомления не используются.

## Данные

Данные MVP хранятся в `data/eighty.json`. Для production лучше заменить JSON-файл на SQLite или PostgreSQL.
