# Eighty v2

Telegram Mini App MVP для контроля питания, воды, веса и прогресса.

## Что входит

- стартовый экран `Дневник`;
- нижнее меню: `Дневник`, `Аналитика`, `Добавить`, `Избранное`, `Профиль`;
- круговой индикатор калорий: съедено, осталось, цель;
- приёмы пищи прямо на главном экране;
- компактная форма добавления еды;
- вода с быстрыми кнопками и ручным вводом;
- БЖУ в виде прогресс-баров;
- вес и прогресс до цели;
- личная база продуктов в разделе `Избранное`;
- профиль с Telegram ID, датой регистрации и сохранением изменений;
- расчёт BMR и КБЖУ по формуле Миффлина — Сан Жеора;
- режимы: похудение с дефицитом 15–30%, поддержание, набор с профицитом 5–15%.

Из v2 удалены напоминания и отдельные шаблоны блюд.

## Запуск

```powershell
npm.cmd run dev
```

Откройте [http://localhost:3000](http://localhost:3000).

## Telegram

Для Telegram Mini App нужен публичный HTTPS URL, который проксирует этот сервер. Если задан `TELEGRAM_BOT_TOKEN`, сервер проверяет подпись `initData`. Серверные уведомления в v2 не используются.

## Данные

Данные MVP хранятся в `data/eighty.json`. Для production лучше заменить JSON-файл на SQLite или PostgreSQL.
